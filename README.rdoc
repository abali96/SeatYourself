== SeatYourself - OpenTable Clone

Bitmaker Labs assignment: "SeatYourself" was the third Rails practical in the course, the first of which was completely tutorial-free.

Concepts reinforced in the development of SeatYourself were:

* Professional Rails application planning (user stories, planning, mockups, modelling, usability)

* Debugging strategies

* Dates/Times in Rails

* ReST
